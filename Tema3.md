# Порівняльний огляд оперативної пам'яті: DDR4 проти DDR5

Цей документ присвячено детальному розбору двох актуальних стандартів оперативної пам'яті (RAM): перевіреного часом DDR4 та нового високошвидкісного стандарту DDR5. Ми розглянемо їх технічні характеристики, архітектурні відмінності та порівняємо конкретні моделі.

---

## 1. Характеристики та порівняння стандартів DDR4 та DDR5

Перехід з DDR4 на DDR5 — це не просто збільшення швидкості, це значна зміна в архітектурі пам'яті, спрямована на підвищення ефективності та пропускної здатності для сучасних багатоядерних процесорів.

### Основні характеристики DDR4
* **Частота:** Стандартні специфікації JEDEC від 2133 МТ/с до 3200 МТ/с (розгінні профілі XMP до 5000+ МТ/с).
* **Напруга:** Стандартна 1.2В.
* **Управління живленням:** Контролер живлення знаходиться на материнській платі.
* **Архітектура каналів:** Один 64-бітний канал на модуль.

### Основні характеристики DDR5
* **Частота:** Стартує від 4800 МТ/с (актуальні моделі 6000-8000+ МТ/с).
* **Напруга:** Знижена базова напруга 1.1В.
* **Управління живленням (PMIC):** Мікросхема управління живленням перенесена безпосередньо на планку пам'яті.
* **Архітектура каналів:** Два незалежних 32-бітних підканали на один модуль.
* **ECC (On-die):** Вбудована корекція помилок на кристалі.


### Таблиця порівняння характеристик

| Характеристика | DDR4 | DDR5 |
| :--- | :--- | :--- |
| **Ефективна частота** | 2133 - 3200 МТ/с (Base) | 4800 - 8400+ МТ/с (Base/OC) |
| **Напруга (VDD)** | 1.2V | 1.1V |
| **Управління живленням** | На материнській платі | PMIC на модулі пам'яті |
| **Макс. об'єм чипа** | 16 Гбіт | 64 Гбіт |
| **Канали на DIMM** | 1 x 64-біт | 2 x 32-біт |
| **Ключ (виріз)** | Зміщений до центру | Зміщений до центру (інша позиція) |

> **Важливо:** Модулі DDR4 та DDR5 фізично несумісні.

## 2. Огляд та порівняння вибраних моделей з магазину

Для практичного порівняння вибрано два популярні комплекти пам'яті (Kit 2x16GB) від виробника **Kingston**.

### Учасник 1: DDR4 — Kingston FURY Beast Black
"Золотий стандарт" для платформ AM4 та LGA1200/1700 (DDR4).

* **Модель:** Kingston FURY Beast DDR4 3200MHz 32GB Kit (2x16GB)
* **Таймінги:** CL16-18-18
* **Напруга:** 1.35V (XMP)

### Учасник 2: DDR5 — Kingston FURY Beast Black
Сучасний вибір для AM5 та LGA1700 (DDR5).

* **Модель:** Kingston FURY Beast DDR5 6000MHz 32GB Kit (2x16GB)
* **Таймінги:** CL36-38-38
* **Напруга:** 1.35V (XMP)
* **Особливості:** AMD EXPO / Intel XMP 3.0, On-die ECC.

### Порівняльний аналіз

Розрахунок реальної затримки (Real World Latency):

$$\text{Реальна затримка (нс)} = \frac{\text{CAS Latency} \times 2000}{\text{Частота (МТ/с)}}$$

| Параметр | DDR4 (Kingston Beast) | DDR5 (Kingston Beast) | Аналіз |
| :--- | :--- | :--- | :--- |
| **Частота** | 3200 МТ/с | 6000 МТ/с | Пропускна здатність DDR5 майже вдвічі вища. |
| **CAS Latency (CL)** | 16 | 36 | Таймінги DDR5 вищі. |
| **Реальна затримка** | **10 нс** | **12 нс** | Абсолютна затримка DDR5 трохи вища, але компенсується швидкістю потоку даних. |
| **Ціна** | Середня | Висока (+40-50%) | DDR5 дорожча через PMIC та нові чіпи. |

### Висновок

1. **DDR4** — ідеально для бюджетних ігрових ПК.
2. **DDR5** — необхідна для робочих станцій та топових збірок на роки вперед.
